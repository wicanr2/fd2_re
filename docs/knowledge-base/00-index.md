## 標註慣例

- **[已驗證]** — 已在原版實檔 / 反組譯交叉確認。
- **[假設]** — 合理推論但尚未驗證，後輪需確認或推翻。
- **[攻略]** — 來自青衫攻略的玩家觀測值，實作時以反組譯為準。
