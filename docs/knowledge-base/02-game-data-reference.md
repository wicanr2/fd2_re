## 4. 戰鬥與經驗值公式（來源：notes.md，小數點以下一律去除）

### 4.1 物理攻擊傷害

```
AP = 攻方AP
DP = 守方DP
暴擊時 DP = 守方DP / 2（取整）
AP = AP × (1 + 攻方地形AP修正率)（取整）
DP = DP × (1 + 守方地形DP修正率)（取整）
最大傷害 = AP - DP
實際傷害 = 最大傷害 × 0.9 ～ 最大傷害 - 1（亂數）
命中率 = (攻方HIT - 守方EV)%
```

### 4.2 劍技攻擊傷害

```
AP = 攻方AP × 劍技加乘率（取整）
DP = 守方DP
最大傷害 = AP - DP
實際傷害 = 最大傷害 × 0.9 ～ 最大傷害 - 1
命中率 = 100%（劍技恆中）
```

### 4.3 法術攻擊傷害

```
最大傷害 = 法術最大傷害 × (1 - 守方魔法抗性)
實際傷害 = 最大傷害 × 0.9 ～ 最大傷害 - 1
命中率 = 法術內定命中率
```

### 4.4 恢復法術

```
恢復力 = 最大恢復 × 0.9 ～ 最大恢復 - 1
```

### 4.5 經驗值公式

| 行為 | 經驗值公式 |
|---|---|
| 攻擊 | (傷害HP / 總HP) × (守方等級 × 守方每級經驗) × (守方等級 / 攻方等級)；致死視同傷害HP=總HP |
| 恢復法術 | (40 / 施法者等級) × Σ (恢復HP / 總HP) × 受法者等級 |
| 傳送術 | 10 × (受法者等級 / 施法者等級) |
| 行動術 | 8 × (受法者等級 / 施法者等級) |
| 魔刃術／魔鎧術／風行術 | 2 × Σ (受法者等級 / 施法者等級) |
| 麻痹術／毒擊術 | Σ (40×9 / 受法者總HP) × (受法者等級 / 施法者等級) |
| 解毒術／祛麻術 | Σ (40×9 / 受法者總HP) × (受法者等級 / 施法者等級) |

### 4.6 其他規則（notes.md「提示」）

- 等級 20 以上可到教會轉職（建議升到滿級 40）；持轉職特殊道具可轉成最高級職業。
- 升級屬性在特定範圍內以亂數決定 → 可用存讀檔刷較佳成長。
- 商店賣出價 = 原價 75 折（3/4）。
- 前 27 章最多 16 人出場，最後 3 章 20 人。
- **裝備上自帶的法術不耗 MP，但施放也得不到經驗值。**（item.md 註）

## 8. 跨文件交叉驗證與衝突

### 8.1 可互相印證（一致）

1. **物品編號一致**：item.md 開頭「商店最強武器/防具」表（2Ah/2Bh/69h/47h/0Ah/0Bh/1Fh/1Dh/68h/33h/3Ch/3Dh/A2h/A3h/9Ah/9Bh/B5h/BFh）與 5.1–5.13 主表的編號、數值**完全吻合**，無衝突。
2. **轉職道具編號一致**：list.md 轉職建議提到的道具（勇者徽章、飛龍卵、聖者之戒、精靈契印、領悟之書、心眼之書、白金徽章）皆對應 item.md 的 59h/CDh/58h/5Ah/5Bh/5Ch/5Dh，名稱與用途吻合。
3. **法術別名一致**：notes.md 的別名表（攻擊術=魔刃術等）與 spell.md 正式名一致，spell.md 採用正式名。
4. **裝備自帶法術與法術參數呼應**：item.md 中「自帶 XX 術（距N範M）」的距離/範圍，多數與 spell.md 的法術距離/範圍相符或為裝備特化版（例：光束砲 4Fh 自帶天火術直線距15範4，是強化版；龍之杖 3Ch 自帶炎龍術距2範0，是縮短版）。重製時須注意：**裝備自帶法術的距離/範圍由裝備定義，不必等於該法術原始值**。
5. **公式自洽**：notes.md 傷害公式中的「實際傷害 = 最大傷害×0.9 ～ 最大傷害-1」與 spell.md 各法術「實際傷害」欄位的數字區間完全吻合（例：天火術最大 500 → 450–499）。可作為公式正確性的交叉驗證。

### 8.2 命名／用字不一致（需統一）

1. **達克賽 / 達可賽**：list.md 成長表（7.2）寫「達可賽」，同檔初始值表與 spell.md 寫「達克賽」。判定為同一角色，重製採「達克賽」。
2. **祭司 / 祭師**：spell.md 用「祭司」，list.md 成長表用「祭師」，item.md 防具「祭師袍（A7h）」。同一職業，建議統一為「祭司」（裝備名「祭師袍」可保留原文）。
3. **聖騎士的魔抗**：7.2 成長表標聖騎士魔抗 10%、龍騎士 0%（升級欄）；但 list.md 轉職建議表（7.5 來源）列聖騎士魔抗 10%、龍騎士 0%，一致。注意龍騎士魔抗 0 而聖騎士 10% 是龍騎士的取捨成本。

### 8.3 編號順序與數值非單調（非衝突，但實作須留意）

- 拳爪系 42h 殭屍爪（AP175）> 43h 碎岩爪（AP150），編號遞增但 AP 不遞增。表示**物品編號不保證按強度排序**，重製不可用編號推算強度。
- 防具 9Ch–A3h（重鎧系）與 90h–9Bh（鎖甲系）DP 區間重疊，分屬不同職業可裝備的甲種。職業×甲種的可裝備對應關係攻略未完整列出，需反組譯補完。
